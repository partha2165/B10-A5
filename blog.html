Important FAQ

What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for the web documents.In an HTML document,DOM allows JavaScript to access and modify the contents of the page.This is how interactive features on websites are implemented.

How do you select an element from the DOM?

There are many ways to select an element

  • getElementById()
  • getElementsByClassName()
  • getElementsByTagName()
  • querySelector()
  • querySelectorAll()

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique in JS that set event propagation (bubbling) to handle events at a higher level in the DOM than the element on which the event originated.Instead of adding individual event listeners to multiple child elements,one can add a single event listener to a parent element.This parent element can then manage all events that bubble up from its child elements.

How do you manipulate an element's attributes and styles using the DOM?

There are many ways to manipulate an element's attributes and styles using the DOM:

  • setAttribute()
  • getAttribute()
  • removeAttribute()

For styling:

One can directly set the 'style' property of an element

One can use the 'classList' property to add, remove, or toggle classes
